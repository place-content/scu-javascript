# TaskFlow UI/UX 디자인 발표 자료

## 개요

TaskFlow는 간단하고 직관적인 개인 업무 관리 애플리케이션으로, 사용자가 효율적으로 할 일을 관리할 수 있도록 도와주는 서비스입니다.

## 디자인 시스템

### 컬러 팔레트

**주요 색상:**
- Primary: #667eea → #764ba2 (그라데이션)
- Success: #28a745
- Info: #17a2b8
- Warning: #ffc107
- Danger: #dc3545
- Light: #f8f9fa
- Dark: #343a40

**카테고리 색상:**
- 개인: #6c757d
- 업무: #007bff
- 학습: #28a745
- 건강: #dc3545
- 쇼핑: #ffc107
- 기타: #17a2b8

**우선순위 색상:**
- 낮음: #d4edda
- 조금 낮음: #cce5ff
- 보통: #fff3cd
- 높음: #f8d7da
- 매우 높음: #f5c6cb

### 타이포그래피

- **폰트:** Noto Sans KR
- **제목:** Bold, 700 weight
- **본문:** Regular, 400 weight
- **캡션:** Light, 300 weight

### 컴포넌트

**버튼:**
- 기본 패딩: 0.75rem 1.5rem
- 둥근 모서리: 8px
- 호버 효과: translateY(-2px)
- 그림자 효과: 0 4px 15px rgba

**카드:**
- 기본 여백: 1.5rem padding
- 둥근 모서리: 8px
- 기본 그림자: 0 2px 10px rgba
- 호버 그림자: 0 4px 20px rgba

**폼 요소:**
- 기본 패딩: 0.75rem 1rem
- 둥근 모서리: 8px
- 포커스 효과: 0 0 0 0.2rem rgba(0, 123, 255, 0.25)

## 반응형 디자인

### 데스크톱 (1200px+)
- 3단 통계 카드 레이아웃
- 2단 할 일 추가/필터 레이아웃
- 최대화된 모달 크기

### 태블릿 (768px - 1199px)
- 1단 통계 카드 수직 정렬
- 할 일 추가/필터 수직 정렬
- 적절한 모달 크기

### 모바일 (576px - 767px)
- 전체 화면 너비 활용
- 수직 스택 레이아웃
- 터치 친화적 버튼 크기

### 스마트폰 (575px-)
- 최소화된 패딩
- 축소된 텍스트 크기
- 세로 방향 최적화

---

## 애니메이션 및 인터랙션

### 페이지 전환
- 페이드 인 효과 (0.3s ease-out)
- 슬라이드 애니메이션
- 부드러운 전환

### 컴포넌트 인터랙션
- 버튼 호버: translateY(-2px)
- 카드 호버: translateY(-2px), 그림자 증가
- 체크박스: 원형 애니메이션
- 토스트 알림: 슬라이드업

### 로딩 상태
- 스피너 애니메이션
- 스켈레톤 로딩
- 진행 표시기

---

## 접근성 고려사항

### 키보드 내비게이션
- 모든 인터랙티브 요소 Tab 키 접근
- 포커스 표시 (outline: none, 배경색 변경)
- 스킵 내비게이션 링크

### 색상 대비
- WCAG 2.1 AA 준수 (4.5:1)
- 색상만으로 정보 전달 금지
- 아이콘과 텍스트 조합

### 스크린 리더
- 적절한 ARIA 레이블
- 시맨틱 HTML 구조
- 이미지 대체 텍스트

### 모션 감소
- prefers-reduced-motion 미디어 쿼리 지원
- 애니메이션 비활성화 옵션

## 사용자 경험 개선 제안

### 단기 개선안
1. **검색 기능 추가:** 할 일 제목/내용 검색
2. **드래그 앤 드롭:** 할 일 순서 변경
3. **푸시 알림:** 마감일 임박 알림
4. **음성 입력:** 모바일 환경에서 음성으로 할 일 추가

### 중기 개선안
1. **칸반 보드:** 프로젝트별 할 일 관리
2. **팀 기능:** 다중 사용자 지원
3. **파일 첨부:** 할 일에 이미지/문서 첨부
4. **데이터 시각화:** 차트와 그래프 제공

### 장기 개선안
1. **AI 추천:** 마감일 예측, 우선순위 추천
2. **캘린더 연동:** Google Calendar, Outlook 연동
3. **오프라인 모드:** PWA 기능 지원
4. **다국어 지원:** 영어, 일본어, 중국어 확장

## 기술 스택

### 프론트엔드
- **HTML5:** 시맨틱 마크업
- **CSS3:** Flexbox, Grid, Custom Properties
- **JavaScript (ES6+):** 비동기 처리, 모듈화
- **Bootstrap 5:** UI 프레임워크
- **Bootstrap Icons:** 아이콘 라이브러리

### 백엔드
- **Node.js:** 서버 환경
- **Express.js:** 웹 프레임워크
- **MongoDB:** 데이터베이스
- **Mongoose:** ODM

### 보안
- **JWT:** 인증 토큰
- **bcrypt:** 비밀번호 해싱
- **CORS:** 교차 출처 요청
- **XSS 방지:** HTML 이스케이프
